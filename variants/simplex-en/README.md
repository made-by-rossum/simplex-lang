# simplex-en (draft)

English-flavored simplex variant.

- Grammar: defined by `spec/framework-v0.1.md`
- This variant provides localized affixes/particles and an English-leaning root lexicon.

## v0.1 choices

- Clause order: SVO
- Adjectives: pre-nominal
- NEGATION: particle `not`
- QUESTION: particle `qa` at clause start

## Affixes (draft)

These realize the shared meaning map for v0.1.

### Number
- PLURAL: noun suffix `-s`

### Tense
- PAST: verb suffix `-d`
- FUTURE: verb suffix `-l`
- PRESENT: unmarked

### Derivation
- AGENT: suffix `-er`
- OPPOSITE: prefix `un-`

## Examples

- `mi see dog` — I see dog.
- `mi seed dog` — I saw dog.
- `mi seel dog` — I will see dog.
- `mi not see dog` — I do not see dog.
- `qa mi see dog` — Do I see dog?
- `mi see dogs` — I see dogs.
- `teach-er` — teacher
- `un-clear` — unclear
