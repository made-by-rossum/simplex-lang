# simplex-lang

A family of simplified, regularized natural-language variants built on a shared, hardened rulebook.

- **Framework name:** `simplex`
- **Example variants:** `simplex-en`, `simplex-fr`, `simplex-es`

## Repo layout

- `spec/` — shared framework specification
- `variants/` — per-variant affixes + root lists
- `examples/` — example texts rendered across variants

## Status

Early draft. We’re starting with a tiny, strict v0.1 spec and expanding from there.
